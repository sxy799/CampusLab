# CampusLab（实验室管理系统）

> 学期PHP课程设计

##  功能

1. 实现不同身份登录实验室管理系统，如系统管理员、实验室管理员、普通用户
2. 系统管理员：开户、销户、查询等。
3. 实验室管理员：增加实验室信息、删除实验室信息、修改实验室信息、查询实验室信息。
4. 普通用户 : 登录、注册、选课、退订、留言。
